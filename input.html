Checkbox.js


Briefing

Please create a js functionality for [type="checkbox"];

Have in mind that:

  • input + label must have a common parent, pretty-checkbox
  • inputs must have ID attribute
  • labels must have FOR attribute, matching input's ID
  • Match label's FOR attribute with matching ID
  • If's
  • If the wrapper doesn't exist, wrap input and label in pretty-checkbox
  • If inputs do not have ID, add incremental ID on each type="checkbox" in the page"
  • If labels do not have FOR attr, add it, matched with corresponsing input
  • If input is placed before label, and both have matching ID and FOR, no js is needed
  • If input is placed before label, but do not have matching ID and FOR, just add ID and FOR
  • If input is placed inside or after the label, check with JS wheter the input is checked or not and add / remove class="checked">on/from label

Please see all edge cases below.

Inputs have ID, Labels have FOR

Inputs DO NOT have ID, Labels DO NOT have FOR
